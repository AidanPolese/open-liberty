    cd petstore
    mvn install
    mvn liberty:start-server

## Contribute to Open Liberty
Our [CONTRIBUTING](https://github.com/OpenLiberty/open-liberty/blob/master/CONTRIBUTING.md) document contains details for submitting pull requests.

1. Clone the repository to your system.

    ```git clone git@github.com:OpenLiberty/open-liberty.git```

2. Run a gradle build.

    ```cd open-liberty/dev```
    
    ```./gradlew cnf:initialize```

    ```./gradlew build```

3. Go [Open issues](https://github.com/OpenLiberty/open-liberty/issues), [Review existing contributions](https://github.com/OpenLiberty/open-liberty/pulls), or [Submit fixes](https://github.com/OpenLiberty/open-liberty/blob/master/CONTRIBUTING.md).
