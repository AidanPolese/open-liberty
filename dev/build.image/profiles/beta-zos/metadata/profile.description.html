The lightweight WAS Liberty for z/OS is production-ready and designed for developers. It is Java EE 7 certified.

Installation

Complete the following steps to install the WAS Liberty:

  1. Download WAS Liberty for z/OS Beta by clicking Download.
  2. Run the following command: cat wlp-beta-zos-@VERSION@.pax | pax -r -ppx
