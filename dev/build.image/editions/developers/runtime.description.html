This basic runtime package includes all you need to get up and running with a Liberty profile server. To keep the footprint small, only a subset of technologies is included. If you need Web Services, JMS, or MongoDB support, also download and install the Extended package in addition to this install.

Installation

There are two ways to download and install the Liberty profile runtime:
  • From within your Eclipse IDE.
  • As a standalone JAR file that you install from the command line.

Installing from within Eclipse

  1. If you don’t already have Eclipse, install Eclipse 4.4.2 for Java EE Developers (Luna SR2).
  2. Select the following Install button , drop it onto the Eclipse toolbar, and then follow the prompts. This installs the WebSphere Developer Tools into your Eclipse installation.
  3. In the Servers view of Eclipse, right-click to define a new Liberty profile server. This opens a wizard to guide you through the process. The wizard includes the option to download and install the runtime as a part of defining a new server.

Installing from the command line

  1. Download @PRODUCT_OWNER@ @PRODUCT_NAME@ V@VERSION@ Liberty profile.
  2. Run the following command: java -jar @ARCHIVE_NAME@.jar
