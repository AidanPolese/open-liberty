Installation

If you need support for Web Services, JMS, or MongoDB, download and install this package on top of your WAS Liberty installation. Please note, this package will only install on a runtime installed using the WAS Liberty V@VERSION@ Runtime Jar package. It will not install on a runtime installed using one of the WAS Liberty zip packages.

  1. Download WAS Liberty Extended Programming Models V@VERSION@ by clicking Download.
  2. Extract the content in the same location as your Liberty installation: java -jar @ARCHIVE_NAME@.jar
